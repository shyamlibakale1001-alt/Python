# Rose's Coffee Shop GUI ☕

A sleek, desktop coffee shop application built with Python and Tkinter. This project transitions a classic terminal-based ordering system into a visual, user-friendly desktop experience.

## ✨ Features
- **Object-Oriented Design:** Built using clean, reusable `Coffee` and `Order` classes.
- **Dynamic Menu:** Automatically generates interactive buttons for menu items.
- **Live Receipt:** Updates your order summary and total cost in real-time as you add drinks.
- **Cozy Coffee Aesthetic:** Styled with a custom, earthy color palette (`#F0DAAE`, `#724816`, etc.) and a modernized flat UI layout.

## 🚀 How to Run
1. Make sure you have Python installed.
2. Clone this repository or download the script.
3. Open your terminal/command prompt and run:

```bash
python coffee_gui.py
